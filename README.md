# 🔐 JA4 Encrypted Traffic Monitor

## 🚀 Features
- TLS traffic capture
- JA4 fingerprint generation
- Application identification
- SQLite database
- Hacker-style dashboard
- Real-time monitoring

## 🛠 Tech Stack
- Python (Scapy, Flask)
- SQLite
- Chart.js (optional)

## ▶️ Run
pip install -r requirements.txt  
python capture.py  
python app.py  

## 🎯 Objective
Identify applications from encrypted traffic without decryption.
